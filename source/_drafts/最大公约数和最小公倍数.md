## 最大公约数求法

### 1. 辗转相除法

```java
public int getMaxCommonDivisor(int a, int b){
    int temp = 1;
    int aa = Math.max(a,b);
    int bb = Math.min(a,b);
    while(temp != 0){
        temp = aa % bb;
        aa = bb;
        bb = temp;
    }
    return aa;
}
```
